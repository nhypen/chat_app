# chat app

Simple chat application made with HTML, CSS and JavaScript.

## Features
- Sending messages
- Auto replies
- Enter to send messages

## How to run the app
1. Download or clone the repository:
git clone https://github.com/nhypen/chat_app.git

2. Open the project folder.

3. Open the file 'index.html' in your browser:
- double click the file
OR
- right click -> open with browser

4. The chat will open and you can start typing messages.
